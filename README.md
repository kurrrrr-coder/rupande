# Rupande 🏛️

Platform website resmi untuk instansi pemerintah atau organisasi. Dibangun dengan HTML, CSS, dan JavaScript vanilla yang responsif dan modern.

## 🎯 Fitur Utama

- ✨ **Responsive Design** - Kompatibel dengan desktop, tablet, dan mobile
- 🎨 **Modern UI/UX** - Interface yang clean dan profesional
- 📱 **Mobile First** - Dioptimasi untuk perangkat mobile
- ♿ **Accessibility** - Mempertimbangkan aksesibilitas pengguna
- 🚀 **Performance** - Loading cepat tanpa dependensi eksternal
- 🔍 **SEO Friendly** - Struktur HTML yang semantic

## 📋 Bagian-Bagian Website

1. **Navbar** - Menu navigasi dengan hamburger menu untuk mobile
2. **Hero Section** - Banner welcome dengan headline dan tagline
3. **Profil Instansi** - Informasi tentang organisasi/instansi
4. **Layanan** - Daftar layanan yang disediakan
5. **Berita** - Berita dan update terbaru
6. **Footer** - Informasi kontak dan copyright

## 🚀 Cara Menggunakan

### Lokal Development
```bash
# Clone repository
git clone https://github.com/kurrrrr-coder/rupande.git

# Buka di browser
# Gunakan Live Server atau buka index.html langsung
```

### Deploy ke Vercel
Website ini sudah siap untuk di-deploy ke [Vercel](https://vercel.com):

1. Push repository ke GitHub
2. Connect di Vercel dashboard
3. Vercel akan auto-deploy setiap kali ada push

Live: https://rupande.vercel.app

## 📝 Customization

### Edit Informasi Instansi
Buka `index.html` dan cari:
- Navbar brand (line 368)
- Hero content (line 384-386)
- Profile section (line 390-405)
- Services dan news (line 408-447)
- Footer contact (line 450-453)

### Edit Warna/Style
Buka `style.css` dan ubah:
- Color scheme
- Font sizes
- Spacing dan padding
- Breakpoints responsive
